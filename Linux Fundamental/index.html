About This CTF

This Capture The Flag (CTF) challenge is designed by UpskillNexus to help you master fundamental Linux skills. Each level presents a real-world scenario that tests your knowledge of Linux commands and system administration.

Practical Learning

Hands-on challenges that mimic real Linux administration tasks

Progressively challenging levels covering all Linux fundamentals

Certification

CTF Challenges

Level 1

File Navigation

You're in a directory with a file named 'secret.txt'. How would you view its contents?

Use the command that displays file contents

Level 2

File Permissions

A script 'runme.sh' isn't executable. What command would make it executable for the owner?

Use chmod with the appropriate permission code

Level 3

Process Management

How would you find all processes running as the user 'www-data'?

Use ps with user filtering

Level 4

Text Processing

How would you count the number of lines in a file called 'data.log'?

There's a command specifically for counting lines

Level 5

Networking

What command would show you all listening TCP ports on your system?

Use netstat or ss with the right options

Level 6

Log Files

How would you check the last 20 lines of the system log file (/var/log/syslog)?

Use a command that can show the end of a file

Level 7

Network Configuration

How would you check the IP address of your system?

There are multiple commands to show network information

Level 8

Text Searching

How would you use grep to search for the word "error" in all .log files in the current directory?

Use grep with a wildcard for file selection

Level 9

User Management

How would you check which users are currently logged in to the system?

There's a simple command that shows logged in users

Level 10

Web Servers

How would you find the installation path of Apache web server?

How would you host a simple Python HTTP server on port 8000?

Directory Creation

File Copying
